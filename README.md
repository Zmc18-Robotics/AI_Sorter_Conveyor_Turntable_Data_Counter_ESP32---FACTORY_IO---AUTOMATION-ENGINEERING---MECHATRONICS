# Description

This is one of the sorting systems that used in a factory, but here we use turntable ( kinda slow ) but it shows you how it works. here we use Esp32 to monitoring the stuffs that entering the factory

using Python and C++

Here we also can combo it with HTML, CSS, and JavaScript if you want it more advanced like IoT monitoring systems and Industrial SCADA

Here the materials :

Factory Hardware and Automation :
- Belt Conveyor (6m) x6
- Curved Belt Conveyor CW x1
- Emitter x1
- Remover x4 ( if you want them, just put them at the end of the simulation )
- Turntable x1
- Capatitive Sensor x1
- Vision Sensor x3 ( Different Color Models )

RL Hardware :
- Esp32 x1
- USB cable x1 ( For the communications )
- LCD (16x2) I2C x1

and some decorations
